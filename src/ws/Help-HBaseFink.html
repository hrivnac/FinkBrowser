hbase special search

The hbase special search panel allows searching connected HBase databases in Fink-specific ways.

  • Area Selector: Searching for alerts within specified ra*dec angular rectancle.
  • Cone Selector: Searching for alerts within specified angular distance from a ra*dec direction.
  • Free Form Selector: Any (Java style) code returning boolean value. It can use any column names (without family name part).
  • othert columns: A coma-separated list of column names (without family names) to be shown. If not specified, only the basic columns and coliumns used in the search are shown.
